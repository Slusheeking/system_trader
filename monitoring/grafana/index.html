Day Trading System - Grafana Dashboards

This page provides access to the Grafana dashboards for monitoring the day trading system.

Portfolio Dashboard

Real-time monitoring of portfolio performance metrics.

  • Portfolio value (total, cash, and assets)
  • Portfolio returns (daily and intraday)
  • Portfolio drawdown
  • Risk metrics (volatility, Sharpe ratio, beta)
  • Trading volume per hour
  • Trade count per hour
View Dashboard

Model Performance Dashboard

Monitoring of machine learning model performance metrics.

  • Model accuracy for different model types
  • Prediction latency (95th percentile)
  • Model versions currently in production
  • Predictions per second
  • Model cache hit rates
  • Detailed model performance metrics
View Dashboard

Dashboard Configuration Files

The dashboard configurations are stored as JSON files in this directory:

  • portfolio_dashboard.json - Portfolio Dashboard
  • model_performance_dashboard.json - Model Performance Dashboard

Importing Dashboards

To import these dashboards into your Grafana instance:

  1. In Grafana, go to Dashboards > Import
  2. Upload the JSON file or paste its contents
  3. Select the appropriate Prometheus data source
  4. Click Import

Metrics

These dashboards rely on metrics exposed by the Prometheus exporters:

  • SystemMetricsExporter: System-level metrics (CPU, memory, disk, network)
  • TradingMetricsExporter: Trading-related metrics (portfolio, orders, trades)
  • ModelMetricsExporter: Model-related metrics (predictions, accuracy, latency)
